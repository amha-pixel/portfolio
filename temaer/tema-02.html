Grundlæggende web

Tema 02

Vi startede med at lære HTML og byggede websitets rygrad, først af rå struktur, derefter med semantik. Fokus var på at bruge de rigtige tags de rigtige steder, så koden ikke endte som én lang identitetskrise af "div's". Jeg arbejdede med links, navigation, genbrug af kode og en hierarkisk mappestruktur, der både giver mening for browsere, undervisere og mit fremtidige jeg.

Se kontruktionen på mit site

CSS: når struktur får stil (og begynder at opføre sig ordentligt)

Herefter blev Cascading Style Sheets introduceret, så mit website begyndte at ligne noget, man har lyst til at bruge. Vi arbejdede med designprincipper som gestalt, kontrast, farvelære, typografi og visuelt hierarki og lærte, hvordan margin, padding og font-valg kan være forskellen på “færdigt” og “hjemmelavet”. Alt blev samlet i studiestartsprøven, hvor vi udviklede et mobile-first website om computere. Her skulle designet fungere på små skærme først og ikke bare presses sammen til sidst. CSS Grid, Flexbox og media queries gjorde layoutet responsivt og for det meste samarbejdsvilligt.

Billeder, gallerier og responsivitet

Jeg fik viden om at arbejde med billeder fra et både teknisk, juridisk og bæredygtigt perspektiv. Det betød billedformater som webP, svg, png og jpg, korrekt størrelse og optimering, så websitet ikke sluger unødvendig båndbredde. Som en del af undervisningen udviklede jeg et billedgalleri, som jeg kunne bruge på mit website om computere, hvor fokus var på layout, struktur og responsivitet frem for at smide billeder ind og håbe på det bedste, og vi fik en grundigere gennemgang af Git og GitHub. Jeg afsluttede forløbet med validering af koden og aflevering via WISEflow med svedige håndflader og begyndende tech neck, men uden panik.
